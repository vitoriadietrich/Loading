# Loading Animation using HTML & CSS | CSS Loading Spinner

Um projeto simples de **animação de carregamento (spinner)** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar um **loading spinner moderno**, ideal para indicar carregamento em páginas web.

## 🛠️ Tecnologias

- HTML
- CSS (animation, keyframes)

## 🎯 Funcionalidades

- Spinner animado
- Design simples e moderno
- Fácil de personalizar

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
